## Violations Found & Fixed

### 1. Raw font in `DesignSystem.swift` — `PreviewListRow.subtitleFont`
- **Line 1288**: `.custom("Inter-Medium", size: 12)` for `.programRoutine` case
- **Fix**: Replaced with `AppFont.caption.font` (identical output, uses token)

### 2. Raw font weight in `DesignSystem.swift` — `SheetHeader`
- **Line 1387**: `.fontWeight(.semibold)` on "Done" button text
- **Fix**: Replaced with `.font(AppFont.label.font)` (Inter-SemiBold 17pt)

### 3. Shadow with raw color in `DesignSystem.swift` — `UnitTabBar`
- **Line 2063**: `.shadow(color: Color.black.opacity(0.02), radius: 8, x: 0, y: -1)`
- **Violation**: Shadow on core navigation element + raw `Color.black`
- **Fix**: Removed shadow entirely (design system specifies contrast separation, not shadows)

### 4-5. Touch targets < 44pt in `CyclesView.swift`
- **Lines 196, 233**: Edit buttons with `.frame(width: 36, height: 36)`
- **Fix**: Changed both to `.frame(width: 44, height: 44)` (HIG minimum)

### 6. Inline font weight in `CyclesView.swift`
- **Line 345**: `.font(AppFont.body.font.weight(.medium))`
- **Fix**: Replaced with `.font(AppFont.body.font)` (body is already Inter-Medium)

### 7. Inline font weight in `CreateCycleView.swift`
- **Line 226**: `.font(AppFont.body.font.weight(.semibold))`
- **Fix**: Replaced with `.font(AppFont.productAction)` (Inter-SemiBold 17pt token)
